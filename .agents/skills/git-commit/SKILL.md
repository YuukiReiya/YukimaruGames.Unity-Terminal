# git-commit

このスキルは、`.agents/git-credentials.txt` の情報を用いてコミットを実行します。
コミットメッセージの内容（英語/日本語）に関わらず、**Agentからの実行結果の報告や提案は必ず日本語**で行ってください。
**リモートへのPushは本スキルの範囲外であり、勝手に実行してはいけません。**

## Usage

変更内容の分析が完了した際に使用します。
実行前に プロジェクトルートにある`.agents/git-credentials.txt` の存在を確認してください。
**相対パスが .agents/... であることを常に意識してください。**
もし存在しない場合は、本スキルを実行してエラーを検知した後、直ちに日本語で状況を説明し、セットアップ用ワークフローへ誘導してください。

## Steps

1. **事前確認**: `.agents/git-credentials.txt` の有無を確認します。情報が取得できない場合は直ちに中断し、日本語で報告してください。
2. **プレフィックスの選択**: 変更内容に基づき、`feat`, `add`, `fix`, `refactor`, `docs`, `chore` から適切なものを選択します。
3. **メッセージの構成**: 変更内容に基づき、適切なメッセージ（英語または日本語）を構成します。
4. 
4. **コマンドの実行**: 特殊文字によるエラーを避けるため、各引数を厳密にクォートして実行します。
```bash
bash .agents/scripts/git-commit.sh "{{prefix}}" "{{subject}}" "{{name}}" "{{email}}"
```
