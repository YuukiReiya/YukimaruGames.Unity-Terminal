void YukimaruGames.Terminal.Domain.Service.CommandRegistry.Register ( string[] unresolvedAssemblyNames,
BindingFlags bindingFlags = BindingFlags )
inline

ランタイムでロード済みと未解決なAssemblyの中から利用できるメソッドを登録.

Parameters
unresolvedAssemblyNames未解決(未ロード)のAssembly名
bindingFlags登録メソッドのバインディングフラグ

注意: このメソッドを呼び出すことで未解決なAssemblyで有効な値のものはAssemblyがロードされます。

* 未解決のAssemblyの参照関係を自動で解決する、即ちそのAssemblyが参照しているAssemblyもロードされる。

* 引数で渡されたAssembly名が既にロードされている場合も考慮。

* Assembly名がnull/不正値なら登録はされない.
